# 🍽️ Velora Restaurant Website

Welcome to **Velora Restaurant Website** – a modern, responsive, and elegant restaurant website built using **HTML5**. This project is designed to provide a clean and user-friendly experience for visitors who want to explore the restaurant, browse the menu, reserve a table, and learn more about the business.

---

## 🌟 Features

- 🏠 Beautiful Home Page
- 👨‍🍳 Meet Our Professional Chefs
- 🍔 Restaurant Menu
- 🖼️ Image Gallery
- 📅 Online Table Reservation Page
- ❓ Frequently Asked Questions (FAQ)
- 🔒 Privacy Policy
- 📜 Terms & Conditions
- 🍪 Cookie Policy
- 📱 Fully Responsive Design
- 🎨 Modern & Clean UI
